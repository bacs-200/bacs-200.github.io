2.2 Creating Hyperlinks

Hyperlinks, created with the anchor tag (``), are what make the web a "web." They connect pages together. The `href` attribute specifies the destination URL. Use the builder below to create your own links.

Generated Code & Preview

<a href="https://www.google.com">Visit Google</a>
Visit Google

Note: A link starting with `http` is an external link. A link to a local file like `about.html` is an internal link.

2.3 Embedding Images

Images bring life to a page. The `` tag is used to embed them. It's a self-closing tag and has two essential attributes: `src` (the image source URL) and `alt` (alternative text for accessibility and SEO).

Good alt text is crucial for screen readers and when images fail to load. Be descriptive!

Generated Code

<img src="https://placehold.co/400x300/0ea5e9/ffffff?text=Character" alt="A cute robot character">
