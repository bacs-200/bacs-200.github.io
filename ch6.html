6.1 Creating Forms for Advanced Search

Forms are the primary way to collect user input. The `

` element wraps your input fields. The `action` attribute specifies where to send the data (like Google's search URL), and the `method="GET"` attribute appends the data to the URL, which is standard for searches.

Themed Google Search Form

<!-- The action points to the submission URL -->
<form action="https://www.google.com/search" method="GET">

  <!-- The 'name' attribute is critical. 
       Google's search parameter is "q" -->
  <input type="text" name="q">

  <button type="submit">Search</button>

</form>
