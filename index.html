MCB BACS 200: Web Development

University of Northern Colorado | Professor Moe Manshad

Chapter 1

Foundations of Web Development

Introduction to GitHub, publishing your first web page, basic HTML structure, and viewing source code.

Chapter 2

Structuring Content with HTML

Using headings, paragraphs, hyperlinks, images, and lists to add meaning and structure to web content.

Chapter 3

Organizing Complex Content

Building on basics with complex organizational elements like data tables and site-wide navigation.

Chapter 4

Introduction to CSS

Introducing CSS for styling pages, covering syntax, selectors, and fundamental styling properties.

Chapter 5

Advanced Styling & Layout

Exploring advanced CSS including custom fonts, advanced selectors, and responsive design for all screen sizes.

Chapter 6

Forms & Interactive Elements

Creating interactive forms to collect user input, using various widgets, and embedding external content.

Chapter 7

Project Management Practices

Covering essential project skills like file organization, code validation, and ensuring professional quality.

Chapter 8

Final Project Application

A capstone guide to applying all learned skills to a real-world project, from research to presentation.
